Maya Projects

Introduction

I have created multiple 3D models using Maya previously. Some of the projects I have done are static objects, while some are animated. From these projects, I have learnt how to use basic tools in Maya to create 3D objects.

Programs Used

  • Maya
  • HitFilm [Video Editing]

Learning Outcomes

I learnt the basics of Maya modelling when creating the static 3D models. In terms of shaping the base models, I used tools like extrude, bevel, multi-cut, merge vertices and more.

I also have experience with using the UV mapping tools in Maya, as well as the animation player. I decided to go even further when experimenting with the character animations, where I used skeletal rigging, skinning, and moving joints using different constraints.
