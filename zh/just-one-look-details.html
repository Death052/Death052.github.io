Just One Look（就看一眼）

簡介

我參加了 2026 年的 Global Game Jam，這次的主題是「面具」（Mask），而《Just One Look》就是我為這次活動製作的遊戲。 我與兩位隊友合作，在 3 天內完成了這款遊戲。《Just One Look》是一款受到《小精靈》（Pac-Man）啟發的策略遊戲，玩家必須在一個封閉的大宅中穿梭，一邊收集光球，一邊躲避敵人。

我主要負責：

  • 編寫整個遊戲的程式，實現所有規劃的功能
  • 技術設計，確保遊戲能在短時間內切實完成
  • 構思簡單且富有創意的敘事故事
  • 敵人機制設計

程式邏輯

遊戲完全是在 Godot 中使用 GDScript 語言編寫的。這是我第一次在如此短的時間內，在 Godot 中完成一款完全做好的遊戲。

狀態機（State machines）被用來控制敵人的行為，由一個主狀態腳本處理多個敵人狀態。 射線檢測（Raycasting）用於偵測玩家，而 NavigationAgent2D 則用於敵人的移動路徑尋找。全域腳本（Global scripts）用於處理遊戲狀態，例如玩家的精神錯亂（insanity）狀態以及存盤點。

著色器（Shaders）也被用來創建自定義的視覺效果，像是 CRT 濾鏡，這些是使用 Godot 的著色器語言（gdshader）製作的。這涉及操作片段著色器（fragment shader）以改變 CRT 濾鏡的顏色，以及操作 UV 頂點來製造 CRT 波浪效果。

機制設計

我主要負責遊戲的敵人機制。我們團隊一致認為關卡應該設計成緊湊的空間，因此我決定給予敵人較大的視野範圍。起初，Game Jam 的「面具」主題只體現在帶有各種面具的敵人設計上。然而，我決定將這個概念也融入到遊戲玩法設計中。

面具敵人

在不使用面具或躲在牆後的情況下，第一種類型的敵人會一直追趕玩家直到玩家離開視線，而第二種類型的敵人則會瞬間殺死玩家。 由於 Game Jam 時間有限，我將同一個概念進行微調並套用在兩種敵人上：當玩家戴上面具時，這兩種敵人角色都會忽略玩家。

影子敵人 + 精神錯亂模式

這種敵人只有在玩家配戴面具時間過長、進入精神錯亂模式時才會出現。它會以玩家的黑色剪影形式出現，並複製玩家的移動方向。這在狹窄的空間中帶來了極大的挑戰，玩家需要規劃好自己的移動路線以避免被逼入絕境。 在精神錯亂模式下，玩家可以使用小刀殺死敵人。此時面具也無法再摘下，使得影子敵人成為玩家唯一的威脅。

敘事設計

雖然 Game Jam 的時限限制了遊戲的敘事深度，但我們仍實現了一個無提示的分支敘事，以此來表現邪教主義的負面影響。

在規劃敘事之前，我們曾想過加入讓玩家在正常和瘋狂狀態之間自由切換的能力。然而，透過在觸發瘋狂狀態後將玩家鎖定在該狀態中，展現了無法脫離邪教的社會壓力。這隱喻了加入邪教往往是一條單行道，一旦陷入就很難離開。

學習成果

這是我第一次參加 Game Jam。雖然我對 Godot 很熟悉，但從未在如此短的時間內製作過一款遊戲。我學會了在遊戲規模上做出妥協，並將精力集中在核心玩法機制的編碼上。我也學會了如何透過討論目標和理解彼此的想法來與隊友進行高效協作。
