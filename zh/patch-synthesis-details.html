基於圖塊的合成

介紹

這個專案是我第一個專注於 Compute Shaders 的 Unity 專案，也是我電腦科本科最終年度專案。 這個專案的主要目標是實現一個基於圖塊的合成演算法，這是一種紋理合成演算法，它接收一個 範例紋理，並生成一個任意大小且看起來與範例紋理類似的新紋理。

使用的程式

  • Unity

演算法邏輯

演算法首先接收一個範例紋理，並將範例紋理分割成小的正方形 圖塊。然後保存每個圖塊的邊界像素，用於計算圖塊之間的相似度。

然後，演算法開始創建新紋理，方法是從範例紋理中隨機選擇一個圖塊， 然後根據現有圖塊的邊界像素添加新圖塊。這個演算法相當複雜， 因此如果您感興趣，歡迎查看我撰寫的完整文件。

學習成果

我透過這個專案學到了很多關於 Compute Shaders 的知識，並理解了 使用 Compute Shaders 如何將繁重的計算工作從 CPU 轉移到 GPU， 顯著提高演算法某些方面的效能。

巧合的是，這是我第一個使用 3D 元素的 Unity 專案（或遊戲專案）。 因此，我學會了如何在 Unity 中使用 3D 射線檢測，以及使用一些 3D 材質。
