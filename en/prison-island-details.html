Prison Island

Introduction

Prison Island is a game I made when I was tasked to demake an existing game in less than 2 months while adding a small twist to its gameplay. The game I chose to demake is called "Grey Scout" from Sockpop, which is a stealth based 3D game where the player's goal is to save prisoners and bring them to the exit while avoiding the guard's line of sight.

I am responsible for:

  • Programming the whole game, implementing all planned features
  • Level Design Planning
  • Brainstorming and implementing card mechanism

Scripting

The game is entirely coded in Unity and therefore uses C# as the scripting language. This is my first attempt creating a 3D game in Unity, and therefore learnt how to use and import 3D assets alongside character animations.

I used state machines to control the behaviour of guards and prisoners. I utilized raycasting specifically for guard detection, and used NavMesh for guard movement near the cages. Unity's Post Processing Layer is used to create atmospheric effects. Animation state blend trees are used for conducting smooth blending animations.

Mechanic Design

I designed a card mechanism as the twist for the game. The player can draw cards while in the game by pressing a button. Each card has unique attributes that both benefit and hinder the player in some way. The card abilities have been rigourously playtested to ensure they are not too overpowered or underpowered, and that they can be used in a variety of different ways. to ensure fair gameplay.

Level Design

I made sure the level design of the game is easy to navigate and understood. I drew a map layout to plan out how the player can hide from guards and navigate the level. This is to ensure appropriate levels of challenge and fun for the player.

Learning Outcomes

As a coding project, I understood how demaking a game is different from creating a game from scratch; requirements for game projects like these are often more strict, similar to how a company would have strict requirements for a project. I also learnt how to think about new game mechanics within a restricted context, which doesn't deviate the game's original concept, yet adds value to the gameplay experience.
