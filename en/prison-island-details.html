Prison Island

Introduction

Prison Island is a game made for a university programming project. I am tasked to demake an existing game in less than 2 months, while adding a small twist to its gameplay.

I chose to demake the game "Grey Scout" from Sockpop. This game involves a lot of stealth elements, requiring the player to hide in bushes to avoid guards, while trying to rescue other prisoners and bringing it back to the ship.

Programs Used

  • Unity

Gameplay Design

The game consist of a single large level with 5 cages containing prisoners. The player is required to go to each cage, free the prisoner within the cage, and bring the prisoner back to the ship. However, a guard is placed within the close region of each cage, and the player must avoid being seen by the guard. If a guard sees the player, it will chase the player. The player can hide in bushes to avoid being seen.

As the twist of the game, I created a mechanism that allows the player to draw cards, each with unique attributes that both benefit and hinder the player in some way.

Learning Outcomes

This is my first time creating a gameplay filled game in 3D, branching away from my 2D projects. I learnt how to use Post Processing Layer to create a more atmospheric environment. I also learnt how to use rigged models and import rigged animations in Unity. I also experimented with NavMeshes in unity for guard movement.

Similar to previous projects, I also had a refresher on using state machines on both Prisoner and Guard Agents, as well as using Raycast for guard detection.
