Stitched.EXE

Introduction

Stitched.EXE is an immersive horror platformer about two Nintendo characters, the Ice Climbers. This game has a gameplay time of about 20 minutes, and the first successful solo game project I have ever released publicly. The game explores both ice climbers trying to survive against a supernatural entity with the appearance of their dead sibling.

I am responsible for:

  • Programming the whole game, implementing all planned features
  • Level Design Planning
  • Creating and expanding on hiding mechanics
  • Game Art

Scripting

The entire game is programmed in C# using the Unity Engine, composed of many scenes joined together. The game has a full UI menu system and a smooth platforming system.

Since this is a solo project, all scriptings within the game is done by me. I analysed and used a feel-good platforming tutorial using gravity and vector manipulation and designed a simple attack enemy system. It is also my first time implementing a dialogue system by utilizing arrays of dialogues. I also utilized state machines for enemies, allowing them to have complex custom behavior patterns.

Game Design

The game does not rely on simple combat for gameplay. Since the two main characters are constantly being hunted by a being of eldrith horror, I wanted to incorporate hide and seek elements within the game to create suspense and helplessness for the characters. There are hiding mechanisms in two parts of the game with slight variation.

The first part introduces a huge floating head blended into the background. The floating head will open its eyes and look around the map within a set time. The second part introduces a monster who walks on parts of the map and can teleport around, with only a rythemic heartbeat to indicate the monster's proximity. In both instance, the player needs to hide behind objects to avoid immediate death.

Art Design

I used Clip Studio Paint to draw all assets within the game. I have referenced colour schemes specifically from the game "Little Nightmares", which has a similar narrative to this game. I used a cooler colour palette for the first part of the game, and a eerie warm colour palette for the second part, both sceneries conveying hopelessness.

The pause menu was designed to be simple with minimum elements, but playtesters said it looked too bland. Therefore I looked up references in UI Database and redesigned it by incorporating the ice climbers into the design and changing its layout. Even some levels have been redesigned to become more visually appealing in later stages.
